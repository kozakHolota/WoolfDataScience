Linear Algebra Cheatsheet

Basics

  • Scalar: A single number.
  • Vector: An ordered list of numbers.
  • Matrix: A rectangular array of numbers.
  • Tensor: A generalization of vectors and matrices to higher dimensions.

Operations

  • Addition: \( A + B \) (matrices must have the same dimensions).
  • Scalar Multiplication: \( cA \) (multiply every entry by \( c \)).
  • Matrix Multiplication: \( (AB)_{ij} = \sum_k A_{ik} B_{kj} \).
  • Transpose: \( A^T \) (flip rows and columns).
  • Inverse: \( A^{-1} \) such that \( AA^{-1} = I \) (only if \( A \) is square and invertible).

Special Matrices

  • Identity Matrix \( (I) \): 1's on the diagonal, 0's elsewhere.
  • Diagonal Matrix: Only diagonal entries are nonzero.
  • Orthogonal Matrix: \( Q^T Q = I \).
  • Symmetric Matrix: \( A^T = A \).

Determinant and Rank

  • Determinant: A scalar value representing the scaling factor of the linear transformation described by the matrix.
  • Rank: The number of linearly independent rows or columns in the matrix.

Vector Spaces

  • Span: Set of all linear combinations of a set of vectors.
  • Basis: A set of linearly independent vectors that spans the space.
  • Dimension: The number of vectors in a basis for the space.

Key Concepts

  • Linear Independence: No vector in the set is a linear combination of the others.
  • Null Space: Set of all solutions to \( Ax = 0 \).
  • Column Space: Span of the columns of \( A \).
  • Row Space: Span of the rows of \( A \).

Eigenvalues and Eigenvectors

  • Eigenvector: A vector \( v \) such that \( Av = \lambda v \).
  • Eigenvalue \( (\lambda) \): The scalar by which an eigenvector is stretched or shrunk.

Important Theorems

  • Invertible Matrix Theorem: A matrix \( A \) is invertible if and only if:
    • \( \det(A) \neq 0 \)
    • \( A \) has full rank.
    • The null space of \( A \) contains only the zero vector.
  • Spectral Theorem: A real symmetric matrix can be diagonalized by an orthogonal matrix.
